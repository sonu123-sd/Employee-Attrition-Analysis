# 📊 Employee Attrition Analysis

## 📌 Project Overview

This project focuses on analyzing employee attrition data to identify key factors influencing employee turnover.
The goal is to help organizations make data-driven decisions to improve employee retention.

---

## 🎯 Objectives

* Identify key reasons behind employee attrition
* Analyze patterns based on age, department, salary, and job role
* Provide insights for HR decision-making

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Jupyter Notebook
* Power BI (for dashboard visualization)
* Excel (for initial data handling)

---

## 📂 Dataset

* The dataset contains employee details such as:
- Id
- Age  
- Department  
- JobRole  
- Gender  
- Attrition  
- MonthlyIncome  
- PerformanceRating  
- YearsAtCompany 

---

## 📊 Analysis Performed

* Data Cleaning
* Handling Missing Values
* Outlier Detection
* Exploratory Data Analysis (EDA)
* Data Visualization

---

## 📈 Key Insights

* High attrition observed in certain departments
* Younger employees show higher attrition rates
* Salary and job role impact employee retention

---

## 📉 Visualizations

* Bar Charts
* Pie Charts
* Histogram
* Box Plot

---

## 🚀 How to Run

1. Clone the repository
